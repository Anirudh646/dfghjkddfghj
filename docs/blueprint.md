# **App Name**: CampusConnect AI

## Core Features:

- AI Query Solver: Allow students to ask open-ended questions about courses, fees, facilities, eligibility, and more using natural language processing to find the most relevant answer, employing an AI tool to choose what info to show.
- Admission Information: Provide access to key details like application deadlines, required documents, and step-by-step guides, with human contact info shown contextually as needed by the AI query tool.
- Contact Directory: List direct contact information for key department personnel.
- FAQ: Offer a searchable database of frequently asked questions.
- Dynamic Content Updates: Enable administrators to easily update information regarding courses, fees, and other relevant data.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to convey trust and knowledge.
- Background color: Light blue (#E1F5FE) for a clean and approachable feel.
- Accent color: Teal (#008080) to highlight important information and calls to action.
- Body and headline font: 'Inter', a sans-serif font providing a modern and accessible reading experience.
- Utilize a set of clean, geometric icons that visually represent different aspects of the application process and campus life.
- Maintain a clean, intuitive layout with a prominent search bar and well-organized content sections to enhance the user experience.
- Incorporate subtle animations and transitions to provide feedback and guide users through the application workflow.
